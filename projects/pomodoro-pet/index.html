Mobile Application

Pomodoro Pet

Flutter / Dart Hive DB Provider State Android / iOS

A gamified productivity tool that replaces internal willpower with external empathy. By linking real-world focus to the well-being of a digital pet, Pomodoro Pet turns the negative feedback loop of procrastination into a nurturing gameplay mechanic.

Built to explore how emotional attachment can drive behavioral change in software design.

> THE_CONCEPT_

[SYSTEM LOG]: Procrastination is a failure of emotional regulation, not time management.

User "Willpower" = LOW.
User "Empathy" = HIGH.

SOLUTION: If you don't work, the cat gets sad. The user studies not for themselves, but to keep "Kato" fed and happy.

Key Features & Engineering

The "Watchdog" Engine

Unlike standard timers, this app actively monitors user behavior. Using Android UsageStatsManager, the app detects if a "Blacklisted" app (e.g., TikTok) is opened during a focus session.


Result: Immediate punishment mechanic (Pet HP loss) creating a high-stakes environment for focus.

State Management (Provider)

The app handles complex, interconnected state logic. A background timer service must sync with the Pet Provider (Hunger/Happiness stats) and the Economy Provider (Gold/Inventory) in real-time, even when the app is minimized.

The Gameplay Loop

  • Study: 25m Focus = 50 Gold + XP.
  • Earn: Economy balanced to require consistent work to afford "High Tier" food.
  • Grow: Leveling up unlocks new 16-bit skins, driving long-term retention.

Interface Design

Custom 8-bit assets and UI components built in Flutter.
