# Pomodoro Pet

A gamified Pomodoro timer app with Tamagotchi-style pet mechanics, built in Flutter.

## Overview

Pomodoro Pet combines productivity techniques with pet care gameplay to create an engaging focus timer experience. Complete Pomodoro sessions to keep your virtual pets happy and unlock new companions.

## Features

- **Pomodoro Timer**: 25-minute focus sessions with 5-minute breaks
- **Pet Care Mechanics**: Virtual pets that thrive when you stay focused
- **Gamification**: Unlock new pets and customize your companions
- **Progress Tracking**: Monitor your productivity streaks
- **Minimalist Design**: Clean, focused interface inspired by pixel art

## Tech Stack

- **Framework**: Flutter (Dart)
- **State Management**: Provider pattern
- **Local Storage**: Shared Preferences
- **Platform**: iOS & Android

## Design Process

1. **Research**: Analyzed existing Pomodoro apps and Tamagotchi mechanics
2. **Wireframing**: Created low-fidelity prototypes in Figma
3. **Pixel Art**: Designed custom pet sprites and animations
4. **Development**: Built using Flutter for cross-platform compatibility
5. **Testing**: User testing with university students and remote workers

## Key Learnings

- **Balance**: Finding the right balance between game mechanics and productivity
- **Motivation**: Understanding intrinsic vs extrinsic motivation in productivity apps
- **Flutter Development**: Building cross-platform mobile applications
- **UX Design**: Creating intuitive interfaces for habit-building apps
