# Mastering Tinkering

Documentation of construction experiments, failures, and rapid prototyping from the Creative Technology master's program.

## Overview

This coursework documents my journey through various making and prototyping challenges. Each experiment explores different materials, techniques, and fabrication methods â€” embracing failure as a learning tool.

## Course Philosophy

"Mastering Tinkering" focuses on:
- **Rapid prototyping**: Building fast, failing fast
- **Material exploration**: Working with diverse materials (wood, electronics, textiles)
- **Documentation**: Recording process, not just outcomes
- **Iteration**: Learning from mistakes and refining designs
- **Hands-on learning**: Practical skills over theoretical knowledge

## Projects Included

### 1. Arduino-Based Installations
Interactive electronics using sensors, actuators, and microcontrollers.

### 2. Laser-Cut Mechanisms
Exploring parametric design and mechanical linkages.

### 3. Wearable Prototypes
Soft circuits and e-textile experiments.

### 4. 3D Printing Explorations
Testing printing techniques, supports, and multi-material prints.

### 5. Physical Computing
Integrating code with hardware for interactive experiences.

## Key Learnings

- **Embrace failure**: Every "failed" prototype teaches valuable lessons
- **Document everything**: Photos, videos, and notes capture the process
- **Time management**: Rapid prototyping requires time-boxing and prioritization
- **Tool proficiency**: Hands-on experience with laser cutters, 3D printers, soldering
- **Material properties**: Understanding how different materials behave

## Tools & Technologies

- **Fabrication**: Laser cutter, 3D printer, CNC router
- **Electronics**: Arduino, Raspberry Pi, various sensors
- **Software**: Fusion 360, Adobe Illustrator, Arduino IDE
- **Materials**: Wood, acrylic, fabric, LEDs, motors
